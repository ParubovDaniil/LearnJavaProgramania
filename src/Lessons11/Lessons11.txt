Урок 11 - Использование примитивных и ссылочных типов данных при вызове метода
        Аргументы метода могут быть
        1.Primitive data types - примитивными типами данных
        2.Reference data types - ссылочные типы данных
        При подстановке reference data type в метод подставляется только его ссылка!

        Итоги:
        1.Passing primitives to method - когда в метод мы вместо параметра пишем примитив
        2.Passing references to method - когда в место параметра мы пишем референс тип данных

        Д.З
        Создайте класс Car с тремя переменными: цвет, мотор и количество дверей. Затем создайте класс CarTest, в
        котором должны быть 2 метода. 1-ый изменяет количество дверей объекта класса Car на количество,
        прописанное в параметре метода. 2-ой принимает в параметры 2 объекта класса Car и меняет их цвета местами.
        Примените оба метода в main и выведите на экран атрибуты всех объектов.
